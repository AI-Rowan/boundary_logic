# boundary_logic

Research and code experiments for decision boundary logic and explainable AI.

## Project goals
- Explore decision boundary representations and clustering
- Prototype explainability workflows in R
- Maintain reproducible experiments and notes

## Repository structure (initial)
- `R/` : reusable functions
- `scripts/` : analysis scripts
- `data/` : local data (not committed unless small + non-sensitive)
- `outputs/` : figures/tables generated by scripts (often not committed)
- `README.md` : project overview

## How to run
1. Open `boundary_logic.Rproj` in RStudio
2. Run scripts in `scripts/` (start with `scripts/01_hello_boundary_logic.R`)
